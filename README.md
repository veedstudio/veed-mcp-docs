# VEED Fabric MCP Documentation

Public documentation for the [VEED Fabric MCP Server](https://www.veed.io) — an AI Talking Video Generator.

## View the docs

**[https://veedstudio.github.io/veed-mcp-docs/](https://veedstudio.github.io/veed-mcp-docs/)**

## Agent Skill

This repository includes a [Claude Agent Skill](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview) in `skills/veed-fabric/`. The skill teaches Claude best-practice workflows for character selection, voice browsing, video generation, and credit management when using the VEED Fabric MCP server.
